Professional audio player for WooCommerce with waveform visualization and pitch control.

== Description ==

YOYAKU Player V3 is a professional audio player plugin designed specifically for WooCommerce music stores. It provides a sleek, ultra-thin (48px) player interface with advanced features.

= Key Features =

* WaveSurfer.js integration for waveform visualization
* Pitch control without affecting playback speed
* Ultra-responsive mobile design
* Deep WooCommerce integration
* AJAX-powered track loading
* Template override system
* Extensive hooks and filters

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu
3. Ensure WooCommerce is installed and activated
4. Use shortcode `[yoyaku_player_v3 product_id="123"]`

== Frequently Asked Questions ==

= Does this require WooCommerce? =

Yes, this plugin is designed to work with WooCommerce products and requires WooCommerce 5.0 or higher.

= Can I customize the player appearance? =

Yes! You can override templates in your theme and use extensive CSS variables for styling.

= Is it mobile responsive? =

Absolutely! The player is optimized for mobile devices with touch controls and responsive design.

= Can I use multiple players on one page? =

Yes, you can have multiple player instances on the same page without conflicts.

= How do I add audio files to products? =

Audio files are managed through the `_yoyaku_playlist_files` product meta field.
